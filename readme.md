# webdevs-craft

## Introduction
Webdevs Page with Craft

## Dependencies
- Node 4.2.x
- Gulp 3.9.0

## Installation
Clone Repo than:

```bash
npm install && bower install && gulp init
```

After the first init, continue your work with 

```bash
gulp
``` 

And work only in the `src` folder.
